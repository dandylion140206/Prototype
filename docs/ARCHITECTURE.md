# ARCHITECTURE

## スクリーンエフェクト

### 構成
- `ScreenEffectStack` が各エフェクトパスを管理する。
- 各エフェクトは ShaderMaterial と State を所有する。
- `ScreenEffectDebugger` は Stack を操作する。

### 実行環境
- 通常レンダラーでは、描画・State・プリセットを利用する。
- ヘッドレス環境では、スクリーンエフェクトを初期化せず無効化する。
- ヘッドレス実行は、エフェクト以外のゲームロジックの起動確認を目的とする。
- エフェクトの見た目とプリセット操作は、通常レンダラー上で手動確認する。
